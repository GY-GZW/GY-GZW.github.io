--介绍--

游戏功能及操作细节模拟了 Windows 7 自带扫雷：

  • 第一步不会触雷，也不会是数字。
  • 支持双键排除操作。
  • 支持自定义模式。
  • 可关闭问号。
  • 在本地记录每种难度下最佳五局成绩，可随时清除。
  • 三种难度和图标颜色供选择。

由于作者自身的技术水平有限，游戏中的计时器并不精确，请将游戏成绩仅作为对内参考。

发现游戏 bug 或者提供建议，欢迎给我发邮件。
